# INSTRUCTIONS  

### 1. Install Stow
```bash
sudo apt install stow
```

### 2. Clone Repository
```bash
git clone https://github.com/Anu-bhav/dotfiles ~
```

### 3. Stow Usage
To simulate how the result will be:

```bash
stow -nv zsh
```

```bash
stow -nv tmux
```

To apply the effect run above command without the -n option

```bash
stow -v zsh
```

```bash
stow -v tmux
```
